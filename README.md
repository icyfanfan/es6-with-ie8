# webpack+babel+ie8踩坑demo

## 运行

- npm install
- npm run server
- 页面在entry目录下


## 目录结构

```
├── README.md  // 读我
├── package.json  
├── gulpfile.js  // gulp入口文件
├── webpack.config.js  // webpack入口文件
├── entry  // 打包后的页面
├── pub  // 打包后的js,css,图片
├── src  // 源码目录
│   ├── assets  // 依赖资源
│   ├── javascript   // js源码目录
```
